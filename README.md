# MSc26 Proteomics

Research project investigating proteomics data analysis

## Project Structure

```
MSc26_Proteomics/
├── data/                          # Data files (not tracked in git)
│   ├── raw/                       # Raw, unprocessed data
│   └── processed/                 # Processed data from preprocessing scripts
├── notebooks/                     # Jupyter notebooks for exploration and analysis
│   ├── 01_eda.ipynb              # Exploratory data analysis
│   ├── 02_eda_li.ipynb           # Additional EDA
│   └── 03_mi_test.ipynb          # Machine intelligence testing
├── scripts/                       # Standalone scripts and pipelines
│   └── preprocess.py             # Data preprocessing script
├── src/                           # Source code and reusable modules
│   ├── styles/
│   │   └── colors.py             # Color palettes and styling utilities
│   └── core/
│       ├── statistical_tests.py   # Statistical testing functions
│       ├── visualization.py       # Visualization functions
│       ├── preprocessing.py       # Data preprocessing utilities
│       ├── classifiers.py         # ML classifiers and pipelines
│       └── evaluation.py          # Model evaluation metrics
├── tests/                         # Unit tests
├── requirements.txt               # Python dependencies
└── README.md                      # This file
```
